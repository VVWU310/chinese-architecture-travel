# 古建行记

一个面向中国古建筑旅行记录的单页网页。首页用地图点位管理旅行地点，详情页展示建筑插图、构件细节、年代信息、观看重点和个人点评。

## 功能

- 地图模式首页，可查看、筛选、新增和删除地点
- 地图点位弹窗支持直接进入详情页
- 默认收录 6 个中国古建筑示例点位
- 详情页包含原创古建测绘风格插图和构件细节图
- 建筑小记包含观看重点、核心细节和现场记录提示
- 现场点评可在浏览器本地保存
- 单 HTML 静态页面，无需构建工具

## 使用方式

直接打开 `chinese-architecture-travel.html` 即可使用。

首页地图使用 Leaflet 和 OpenStreetMap 在线资源，查看地图底图时需要网络连接。
